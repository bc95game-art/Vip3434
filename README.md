# 🔥 پنل خودکار V2Ray - لینک اشتراک دائمی

این پروژه یک سیستم خودکار برای جمع‌آوری، پردازش و تولید لینک اشتراک V2Ray است.

## ✨ ویژگی‌ها
- 🤖 **کاملاً خودکار** - هر ۴ ساعت یکبار کانفیگ‌ها به‌روز می‌شوند
- 🔗 **لینک دائمی** - لینک اشتراک شما همیشه ثابت و قابل استفاده است
- 🌐 **صفحه وب اختصاصی** - با فعال کردن GitHub Pages می‌تونید لینک خود را در یک صفحه وب ببینید

## 🚀 نحوه استفاده
1. این ریپازیتوری را Fork کنید
2. در فایل `scripts/collector.py` لینک سابسکریپشن خود را جایگزین کنید
3. GitHub Pages را روی شاخه `main` و پوشه `public` فعال کنید
4. لینک اشتراک نهایی: `https://raw.githubusercontent.com/YOUR_USERNAME/YOUR_REPO/main/configs/subscription.txt`
5. صفحه وب: `https://YOUR_USERNAME.github.io/YOUR_REPO/`
